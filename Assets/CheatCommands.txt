БЕРЕГИТЕСЬ ЛИШНИХ ПРОБЕЛОВ

set_lang 0/1 (english/русский)
cockroach_mode 0/1 (выключить/включить читы)

СЛЕДУЮЩИЕ КОМАНДЫ РАБОТАЮТ ТОЛЬКО В РЕЖИМЕ ЧИТОВ:

set_scene N - где N номер сцены
get_all_scenes - пишет список сцен
restart_scene - перезагружает сцену
exit - закрывает игру


НИЖЕСЛЕДУЮЩИЕ КОМАНДЫ ПОКА НЕ РАБОТАЮТ
get_player - получает объект игрока на сцене БЕЗ ЭТОЙ КОМАНДЫ НИЖЕСЛЕДУЮЩИЕ НЕ РАБОТАЮТ
(Вызывать команду надо будет каждый раз при смене/перезагрузке сцены)

move_to_pos X Y - смещает игрока в указанную позицию
move_on_vector X Y - смещает игрока на указанный вектор
